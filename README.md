# ClipCut — เว็บตัดช็อตเด่นจาก YouTube อัตโนมัติ

วางลิงก์ YouTube → ดึง transcript → ให้ AI เลือกช่วงเด่น → ตัด+ต่อคลิปด้วย ffmpeg

## สิ่งที่ต้องติดตั้งก่อน (บนเครื่อง/เซิร์ฟเวอร์ของคุณ)

1. **Node.js** (v18+) — https://nodejs.org
2. **yt-dlp** — โปรแกรมดาวน์โหลดวิดีโอ/ซับจาก YouTube
   ```bash
   pip install yt-dlp
   # หรือ
   brew install yt-dlp   # mac
   ```
3. **ffmpeg** — โปรแกรมตัด/ต่อวิดีโอ
   ```bash
   brew install ffmpeg   # mac
   sudo apt install ffmpeg   # linux
   ```
4. **Anthropic API key** — สมัครได้ที่ https://console.anthropic.com

## วิธีติดตั้งโปรเจกต์

```bash
cd ssemble-clone
npm install
cp .env.example .env
# แล้วเปิดไฟล์ .env ใส่ ANTHROPIC_API_KEY ของคุณ
```

## วิธีรัน

```bash
npm start
```

แล้วเปิดเบราว์เซอร์ไปที่ `http://localhost:3000`

## วิธีใช้งาน

1. วางลิงก์ YouTube ในกล่อง
2. เลือกจำนวนช็อตเด่นที่อยากได้ และความยาวสูงสุดต่อช็อต
3. กด "ตัดคลิปให้ฉัน"
4. รอประมวลผล (ขึ้นอยู่กับความยาววิดีโอ ปกติ 1-3 นาที)
5. ดูตัวอย่าง + ดาวน์โหลดคลิปผลลัพธ์ได้เลย

## ข้อจำกัดสำคัญที่ควรรู้

- **ต้องมีซับไตเติล (auto-caption หรือซับที่มีอยู่)** ระบบใช้ transcript ในการวิเคราะห์ว่าช่วงไหนน่าสนใจ ถ้าวิดีโอไม่มีซับเลยจะตัดให้ไม่ได้ (เวอร์ชันถัดไปสามารถเพิ่ม speech-to-text ของตัวเองได้ถ้าต้องการ)
- **เรื่องลิขสิทธิ์**: การดาวน์โหลด/ตัดวิดีโอ YouTube ของคนอื่นโดยไม่ได้รับอนุญาตอาจขัดต่อข้อตกลงการใช้งาน (ToS) ของ YouTube และกฎหมายลิขสิทธิ์ แนะนำให้ใช้กับวิดีโอของตัวเอง หรือวิดีโอที่มีสิทธิ์ใช้งานเท่านั้น ถ้าจะเปิดเป็นเว็บสาธารณะ ควรมีข้อตกลงการใช้งาน (Terms of Service) ให้ผู้ใช้ยืนยันว่ามีสิทธิ์ในวิดีโอที่ใส่เข้ามา
- ระบบนี้รันแบบ **synchronous** (รอจนเสร็จ) เหมาะกับการทดสอบ ถ้าจะใช้งานจริงกับผู้ใช้หลายคนพร้อมกัน ควรทำเป็น queue/background job (เช่น BullMQ + Redis) แทน

## แนวทางต่อยอด

- เพิ่มการตรวจจับ scene change ด้วย ffmpeg เพื่อช่วยเลือกจุดตัดที่ภาพไม่กระตุก
- เพิ่มการใส่ subtitle/แคปชั่นลงในคลิปผลลัพธ์อัตโนมัติ
- ทำเป็น queue system รองรับผู้ใช้พร้อมกันหลายคน
- เพิ่มตัวเลือก aspect ratio (9:16 สำหรับ TikTok/Reels)
